簡介

OpenCart

OpenCart為一免費開源電子商務平台，供商家建立其網上商店。 OpenCart為建立一個成功的網上商店提供專業且可靠的基礎。OpenCart擁有大量的功能，可以讓您牢牢掌控自定義的商店，使用方便的界面吸引了第一次在網上開始業務的店主可以輕鬆掌握；強大的功能則吸引了不同國家且擁有豐富經驗的網頁編程師來開發更出色的模組，OpenCart所提供的基礎吸引了各式各樣的用戶。透過OpenCart，可以令您的網上商店充分發揮潛力。

OpenCart 使用指引

本指引為使用OpenCart界面時的導航資源。以下將會詳細介紹設置商店所涉及的資訊：有關更新到最新版的說明、系統 / 伺服器環境的需求、訪問管理界面和卸載。用戶指引涵蓋管理界面的基本功能。我們將引導您瀏覽管理界面的重要部分：目錄、模組、銷售、系統和報告。我們將介紹每個項目所控制的店面部分，以及在管理界面進行改動以滿足商店需求的方法。本指引包含了重要的開店步驟，讓您清楚如何添加商品到您的網店，追踪銷售狀況，管理您的客戶群，更改商店的佈局，添加功能模組的做法等。

閱讀本指引後，您將會更熟悉於OpenCart裡設置、維護及更新您的網上商店。
